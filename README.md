
# TRACE (Tracking & Real-time Awareness of Carbon Emissions)

TRACE is an application designed to help users track and monitor carbon emissions in real-time, providing actionable insights for environmental awareness.

## 🚀 Overview

TRACE leverages modern web technologies and AI to bridge the gap between complex carbon data and user-friendly tracking. This project enables users to understand their environmental impact and make more informed decisions about their carbon footprint.

## 🛠 Tech Stack

* **Framework:** [TypeScript](https://www.typescriptlang.org/) / [Vite](https://vitejs.dev/)
* **AI Integration:** [Google Gemini API](https://ai.google.dev/)
* **Runtime:** Node.js
* **Deployment:** Vercel

## 📦 Getting Started

### Prerequisites

Ensure you have [Node.js](https://nodejs.org/) installed on your local machine.

### Installation & Setup

1. **Clone the repository:**
```bash
git clone https://github.com/nipuntamore/TRACE.git
cd TRACE

```


2. **Install dependencies:**
```bash
npm install

```


3. **Configure Environment Variables:**
Create a `.env.local` file in the root directory and add your Gemini API key:
```env
GEMINI_API_KEY=your_actual_api_key_here

```


4. **Running Locally:**
Start the development server:
```bash
npm run dev

```


Open [http://localhost:5173](https://www.google.com/search?q=http://localhost:5173) (or the port specified in your terminal) to view the app.

## 💡 How to Use

1. **Input Data:** Provide the required information regarding your activities or consumption to the application.
2. **Analysis:** The application uses AI to process this data and calculate your carbon impact.
3. **Review:** Visualize your emissions data and gain insights on potential improvements.
